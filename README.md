## Project Title

Security Assessment and Hardening of a Small AWS Cloud Environment

## Project Description

This project focuses on reviewing and improving the security
of a small AWS cloud environment.

The project covers:

- Security Groups
- AWS IAM
- EC2 Key Pair Management
- Amazon S3 Permissions
- Security Recommendations

## Objectives

1. Review AWS Security Groups.
2. Review IAM users, groups, roles and policies.
3. Review EC2 key pair management.
4. Review S3 permissions.
5. Identify security risks.
6. Apply security improvements.
7. Document security findings.
8. Store project documentation in GitHub.
